# 论文结构与写作指南

## 标准结构

### 摘要 (Abstract) — 250-300字

一个段落，包含：
1. 研究问题（1句）
2. 方法与数据（1-2句）
3. 核心发现（2-3句）
4. 政策启示（1句）

中英文双语摘要（如投中文期刊）。

### 1. 引言 (Introduction) — 1500-2000字

段落结构：
- P1: 研究背景与动机（为什么重要，吸引读者）
- P2-3: 现有文献的空白（不是综述，而是突出 gap）
- P4: 本文做了什么（研究设计概述）
- P5: 核心发现摘要
- P6: 贡献总结（对文献/政策的贡献）
- P7: 论文路线图（各节安排）

### 2. 制度背景 (Institutional Background) — 1000-1500字

- 政策/制度的详细描述
- 关键时间线
- 制度细节（对理解识别策略有帮助的细节）
- 理论预测（不同理论的预期）

### 3. 文献综述 (Literature Review) — 1500-2000字

组织方式（按主题，不按时间）：
- 第一支文献：[与因变量/现象相关的文献]
- 第二支文献：[与自变量/政策相关的文献]
- 第三支文献：[与方法/识别策略相关的文献]
- 本文定位：与以上文献的区别和联系

### 4. 数据与变量 (Data and Variables) — 1000-1500字

- 数据来源和获取方式
- 样本构造过程
- 变量定义（可用表格）
- 描述性统计（Table 1）
- 关键变量分布讨论

### 5. 实证策略 (Empirical Strategy) — 1000-1500字

- 识别策略完整说明
- 估计方程（含数学公式）
- 各项说明
- 识别假设讨论
- 为什么这个策略可信

### 6. 实证结果 (Results) — 2000-3000字

6.1 主回归结果
- 主要结果解读（经济显著性 + 统计显著性）
- 逐步加固定效应/控制变量

6.2 识别检验
- 平行趋势/事件研究图
- 预处理系数联合检验

6.3 稳健性检验
- 安慰剂检验
- 替代定义
- 其他稳健性

6.4 异质性分析
- 分样本结果
- 交互项结果

6.5 机制分析（如有）
- 中介效应
- 调节效应

### 7. 结论 (Conclusion) — 800-1000字

- 核心发现总结（不重复引言）
- 政策含义
- 研究局限
- 未来研究方向

### 参考文献
- 至少 25 篇
- 中文期刊用 GB/T 7714 或期刊要求
- 英文用 APA 7

## 写作风格规则

### 必须遵守

1. **全段落散文**: 正文中不使用项目符号列表。数据部分的表格除外
2. **因果语言谨慎**: 只有在识别策略可信时才用因果语言（"导致"、"引起"），否则用"相关"、"关联"
3. **数据支撑**: 每个论点必须有数据或文献引用
4. **段落充实**: 每个主要章节至少 3-4 个实质性段落（每段 150-250字）
5. **图表完整**: 每个图表有编号、标题、注释（注明标准误类型、显著性说明等）

### 中文特殊规则

- 使用学术经管期刊语调：严谨专业但不故作高深
- 不用"取决于"
- 不过度使用破折号（看起来像 AI 生成）
- 不用"值得注意的是"、"综上所述"、"毋庸置疑"等套话
- 不用感叹号
- 英文术语首次出现时括号标注，之后可直接使用中文

### 回归表格式

```markdown
|---|---|---|---|---|

| | (0.012) | (0.011) | (0.011) | (0.010) |
| 控制变量 | 否 | 是 | 是 | 是 |
| 个体固定效应 | 是 | 是 | 是 | 是 |
| 时间固定效应 | 是 | 是 | 是 | 是 |
| 省份×年份固定效应 | 否 | 否 | 是 | 是 |
| N | 12,345 | 12,300 | 12,300 | 11,890 |
| R² (within) | 0.123 | 0.145 | 0.156 | 0.162 |

注: 括号内为[聚类层级]聚类稳健标准误。* p<0.1, ** p<0.05, *** p<0.01。
```

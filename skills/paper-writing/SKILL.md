# paper-writing — 论文撰写技能指南

适用场景：撰写实证经济/管理学论文全文或任意章节。触发关键词：写引言、写文献综述、写实证策略、写结果讨论、写结论、润色论文、academic writing。

---

## 1. IMRaD 结构规范

标准结构及字数/段落建议（中文经管期刊，全文约 15,000–25,000 字）：

| 章节 | 中文字数 | 英文字数 | 段落数 | 核心任务 |
|------|---------|---------|-------|---------|
| 摘要 | 300–400 | 250–300 | 1 结构段 | 背景+问题+方法+发现+意义 |
| 引言 | 2000–3000 | 1500–2000 | 5–7 | 漏斗式，见下节 |
| 制度背景 | 1500–2500 | 1000–1500 | 4–6 | 外生变异来源，支撑识别假设 |
| 文献综述 | 2000–3000 | 1500–2000 | 5–8 | 按主题组织，见 literature-review 技能 |
| 数据与变量 | 1500–2000 | 1000–1500 | 4–5 | 数据来源+样本描述+变量定义+描述性统计 |
| 实证策略 | 1500–2000 | 1000–1500 | 4–5 | 模型+识别假设+鲁棒性计划 |
| 结果 | 3000–4000 | 2000–3000 | 7–10 | 主结果+事件研究+异质性+稳健性 |
| 结论 | 1000–1500 | 800–1000 | 4–5 | 发现总结+贡献+政策含义+局限性 |
| 参考文献 | — | — | ≥ 25 条 | — |

## 2. 引言写作模板（漏斗式）

**结构逻辑：大背景 → 研究空白 → 本文做法 → 主要发现 → 贡献声明**

### 段落 1：宏观背景与现象
打开引言，用2–3句话引出研究主题的社会/经济重要性。引用权威机构数据或近期政策背景。不使用"随着……的不断发展"等套语。

> 示例框架：
> "[主题] 是 [领域] 中的核心议题。[量化背景数据]。然而，关于 [具体机制/效应] 的因果证据仍然匮乏。"

### 段落 2：文献现状与空白
综述现有研究做了什么，指出三类常见空白：
- 因果识别不足（相关性≠因果）
- 中国/发展中国家情境缺失
- 特定子人群/机制未被探索

> 示例框架：
> "已有研究主要从 [维度A] 和 [维度B] 探讨这一问题（Author, Year; Author, Year），但存在两方面不足：其一……其二……"

### 段落 3：本文做法（识别策略）
一句话介绍数据来源，一句话介绍识别策略，说明外生变异来源。

> 示例框架：
> "本文利用 [数据来源，时间范围，观测单元]，以 [政策/事件] 作为准自然实验，采用 [方法] 识别 [X] 对 [Y] 的因果效应。[政策] 的实施在时间上具有外生性，原因在于……"

### 段落 4：主要发现
先报告基准结果（系数方向+显著性+经济量级），再提1–2个异质性发现。使用具体数字，不使用"显著提高"等无量级的表述。

> 示例框架：
> "基准回归显示，[X] 使 [Y] 平均 [上升/下降] [X]%（[显著性水平]）。进一步分析发现，该效应在 [子样本] 中更为突出，幅度达 [Y]%，而在 [子样本] 中效应不显著。"

### 段落 5：贡献声明
列出2–3点贡献，与文献钩挂，使用"本文贡献于……的文献"的标准学术表达。

**标准贡献三类型：**
1. 文献贡献：首次提供……的因果证据
2. 方法贡献：采用……解决了……的识别难题
3. 政策贡献：发现对……政策制定具有含义

## 3. 文献综述组织方式

**原则：按研究主题/争论点/方法组织，而非按时间顺序列举。**

### 推荐结构（3–4个主题组团）

**主题组A** — 与本文因变量/现象直接相关的文献
> 一段综述该主题的主要发现，指出共识与争议，最后一句引出本文与此文献的关系。

**主题组B** — 与本文自变量/政策/干预相关的文献
> 同上结构。

**主题组C** — 与本文方法/识别策略相关的文献
> 说明本文方法的学术渊源。

**过渡句示例：**
> "本文在上述文献基础上，进一步……"
> "与 [Author, Year] 不同，本文……"
> "本文的贡献在于首次……"

### 综述段落内部结构
1. 开篇句：该组文献研究什么
2. 主要发现综述（引用3–5篇代表文献）
3. 现有局限（1–2句）
4. 本文如何推进

## 4. 识别策略写作要点

### 4.1 基准模型呈现

先用文字说明逻辑，再写方程，再解释符号。

**方程格式（LaTeX）：**
```latex
Y_{it} = \alpha + \beta \, \text{Treat}_{i} \times \text{Post}_{t} + \gamma X_{it} + \mu_i + \lambda_t + \varepsilon_{it}
```

**符号表说明顺序：** 因变量 → 核心解释变量 → 控制变量 → 固定效应 → 误差项

### 4.2 识别假设讨论

每个假设用一段独立讨论，结构：
1. 假设陈述（一句话）
2. 直觉解释（为什么该假设成立）
3. 对应检验（用什么结果支撑）

**DID 核心假设清单：**
- 平行趋势假设 → 事件研究图，预处理系数不显著
- 无预期效应假设 → 领先期系数检验
- 稳定性单位处理假设 (SUTVA) → 讨论溢出可能性
- 无同期混淆政策 → 制度背景节说明

### 4.3 稳健性检验写作

每类稳健性检验用一句话说明目的，一句话说明结果：
> "[检验名称]：为排除……的担忧，我们……。结果（Table X, Column Y）显示系数与基准估计统计上无显著差异。"

## 5. 结果讨论写作要点

**写作顺序：陈述结果 → 统计显著性 → 经济量级 → 机制解释 → 与文献对比**

### 主结果段落模板
```
Table [X] 报告了基准回归结果。列（1）仅控制个体和时间固定效应，
核心变量 [Treat×Post] 的系数为 [X]（标准误 = [Y]），在 [Z]% 水平上显著。
列（2）加入控制变量后，系数变为 [X']，幅度变化约 [%]，说明
[控制变量对选择偏误的纠正是有限/显著的]。从经济量级看，
该系数意味着……相当于样本均值的 [%]。这一发现与 [Author, Year] 的结论
[一致/不同]，[一致原因/差异原因分析一句]。
```

### 异质性分析写作

先说分组依据的经济逻辑，再报告各组系数，再解释为什么存在差异。
> "我们预期该效应在……条件下更强，原因在于……。Table X Panel B 证实了这一预期：……"

### 稳健性结果写作
不要单独用一段列举所有检验，每种检验只需1–2句。
> "（1）安慰剂检验。（2）改变聚类层级。（3）剔除……样本。结果均与基准估计一致（Table X）。"

## 6. 结论写作要点

**禁止：重复引言第一段的套话；重复逐一列举每个回归结果。**

### 结构

**段落1：核心发现（综合，非列举）**
以1–2句话用研究问题语言（非统计语言）表达核心发现。

**段落2：机制与贡献**
本文结果从什么角度推进了对该现象的理解，与现有文献如何衔接。

**段落3：政策含义**
具体，避免空泛。说明"给谁的政策建议，做什么，为什么"。

**段落4：局限性与未来研究**
诚实列出1–2个主要局限（数据限制、外部效度等），提出未来研究方向。

## 7. 中文经管期刊写作风格

### 行文原则
- 专业、严谨，但句子不晦涩，避免超长从句（单句不超过60字）
- 每段有明确主题句，段尾收束
- 数字：中文正文用阿拉伯数字（统计量、年份、百分比）
- 专有名词首次出现时给出英文原名及缩写，之后使用缩写

### 段落长度
- 正文段落：150–300字（不含引用）
- 过短段落（<80字）合并；超长段落（>350字）拆分

### 引言注意事项
- 第一句话不用"随着"、"在……背景下"、"近年来"
- 不用感叹号
- 第一自然段内必须有至少一处实证数据引用

## 8. 禁止用语清单

以下表达在学术论文正文中一律禁止使用：

| 禁止表达 | 替代方案 |
|---------|---------|
| 值得注意的是 | 直接陈述该事实 |
| 综上所述 | （结论段首句直接总结核心发现） |
| 取决于 | "在…条件下" / "随…变化" |
| 不难发现 | 直接陈述 |
| 由此可见 | 直接陈述 |
| 进一步地 | （用具体的逻辑连接词，如"为检验……"） |
| 相关性极强 | 报告具体系数/R² |
| 显著提高/下降 | 报告具体系数和显著性水平 |
| 随着……的不断深入 | 改写为具体背景句 |
| 充分说明了 | 直接说结果说明了什么 |
| （过度使用破折号）—— | 改用逗号或拆句 |
| ！（感叹号） | 删除，换陈述句 |

## 9. 双格式输出规范

### LaTeX 格式（投稿/终稿）

文件结构遵循 `empirical-pipeline/templates/paper-latex.tex`。

关键格式规则：
- 图表用 `\begin{table}[htbp]` + `\centering` + `\caption{}` + `\label{}`
- 三线表：`\toprule`, `\midrule`, `\bottomrule`（booktabs 包）
- 注释行：`\multicolumn{N}{l}{\footnotesize Note: ...}`
- 公式编号：`\begin{equation}\label{eq:main}`
- 引用：`\citep{}` / `\citet{}`（natbib 包）

### Markdown 格式（草稿/中间输出）

- 标题层级：`#` 章，`##` 节，`###` 小节
- 公式：`$inline$` 或 `$$display$$`
- 表格：标准 Markdown 表格，表头加粗
- 图表引用：`Figure X`/`Table X`（占位符）
- 注释：`> **注：**` 格式

### 输出要求

生成正文时，**默认输出 Markdown**（可直接在对话中查看）。
若用户要求 LaTeX，切换为完整 `.tex` 代码块，包含 `\documentclass` 到 `\end{document}`。

## 10. 快速写作检查清单

在输出任何章节前，确认：

- [ ] 每段有明确主题句
- [ ] 无禁止用语
- [ ] 所有实证数字有来源（回归表/描述性统计表）
- [ ] 因果语言谨慎：仅在识别策略可信时使用"导致"/"因果"
- [ ] 文献引用格式一致
- [ ] 段落长度适中（150–300字）
- [ ] 全文结构完整（无"待补充"占位符）
- [ ] 摘要涵盖5要素：背景/问题/方法/发现/意义
